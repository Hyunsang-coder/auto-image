An open-source studio that turns raw app screenshots into polished, multi-language App Store sets: device frames, headlines, highlights, per-locale captions, and pixel-exact PNGs for App Store Connect — entirely in your browser.

Everything between raw screenshot and store listing

Canvas editor

Device frames for iPhone & iPad, headlines and captions with full typography, magnified highlight loupes, badges, ornaments, multi-slide templates, and App-Store-style 2-page panoramas.

Localization built in

Captions live in a slide × language table. Export it as CSV/JSON, translate it in any AI chat or spreadsheet, re-import. Swap per-language screenshots by filename and preview any locale on the canvas.

Store-ready export

Every slide renders at the exact App Store Connect resolution, alpha-stripped so uploads never bounce, zipped as locale/device/NN.png — or as a ready-to-run fastlane deliver folder.

AI-agent friendly

A documented import manifest lets an AI assistant draft your whole screenshot project as files, and a headless pipeline renders it to final PNGs without a single click.

Four steps, one sitting

  1. Set up — pick device and slide count, or import an AI-drafted project in one file selection.
  2. Design — drop in screenshots, write headlines, frame, highlight, decorate.
  3. Localize — export the caption table, translate anywhere, re-import; add per-language screenshots.
  4. Export — one click renders every slide × language at exact store sizes into a ZIP.
