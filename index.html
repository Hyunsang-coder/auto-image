What your AI agent handles

Drafts the whole project

Your agent writes the slides and captions as plain files. One import turns them into a complete, editable project — frames, text, screenshots, every language.

Renders and fixes itself

A headless pipeline exports each slide at the exact App Store size, inspects the layout, and re-runs until it's clean — without a single click.

Every language by default

Captions live in one slide × language table. The agent fills each language; per-locale screenshots swap in by filename.

Yours to fine-tune

Open the canvas editor for frames, highlight loupes, badges, ornaments, and App-Store-style 2-page panoramas whenever you want hands-on control.

From prompt to store, mostly hands-off

  1. Ask your agent — it drafts the slides and captions as files.
  2. Import — one file selection assembles the whole project.
  3. Render — the headless pipeline exports and self-corrects every language at exact store sizes.
  4. Tweak, optionally — open the editor for any polish, then export the ZIP.
